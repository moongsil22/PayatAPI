페이앳 제휴사 API 

본 API는 제휴사 전용 API입니다. 혹시 "오픈 API"를 찾으신다면 다른 링크를 이용해주세요.

--------------------------------------------------------------------

제휴사 API의 특징

1. OAuth 2.0에서 code인증 방식과 사용자 authorize부분을 생략하여 간편한 인증을 채택합니다.
2. 서버 대 서버 통신에 이용되는 API방식이며 , ERP나 기타 고객사 시스템에 Integration하는 것을 목적으로 합니다.
